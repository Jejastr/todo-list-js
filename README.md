# Simple To-Do List

Мій навчальний проект списку завдань, створений на HTML, CSS та JavaScript.

### Функціонал:
* Додавання задач у список.
* Валідація (не можна додати порожню задачу).
* Додавання через кнопку або клавішу **Enter**.


### Технології:
* HTML5
* CSS3 (Flexbox, Transitions)
* JavaScript (DOM Manipulation)
